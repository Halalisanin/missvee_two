## Product Description Framework

Each product description follows this structure:

1. **The Scent Experience** — Sensory description (how it makes you feel)
2. **Scent Notes** — Top, Middle, Base (visualised as a calming graphic)
3. **The Ritual** — How to use this product as part of a self-care routine
4. **Key Features** — Natural ingredients, long-lasting, etc.
