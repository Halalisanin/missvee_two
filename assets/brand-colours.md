# Miss Vee Perfume — Concept Two: Brand Colour Palette

*Inspired by Sanctuary Spa's calming, botanical aesthetic.*

| Role | Colour | Hex Code |
|------|--------|----------|
| Primary Background | Soft Cream | `#FDF8F0` |
| Secondary Background | Calming Sage | `#D4E0D4` |
| Primary Text | Deep Forest | `#2D3A2D` |
| Primary Accent (Buttons) | Sanctuary Orange | `#E87A4A` |
| Secondary Accent | Copper/Gold | `#C9A87C` |
| Success/Messages | Soft Teal | `#6B9B8A` |
| Borders/Lines | Warm Grey | `#E8E0D5` |

## Usage Notes

- **Sanctuary Orange (#E87A4A)** is the primary CTA colour — used for buttons, links, and key interactive elements
- **Copper/Gold (#C9A87C)** is used for premium accents — borders, highlights, and decorative elements
- **Soft Cream (#FDF8F0)** is the main page background — warm and natural like paper or linen
- **Calming Sage (#D4E0D4)** is used for secondary backgrounds — cards, sections, and dividers
- **Deep Forest (#2D3A2D)** is used for all body text and headings — grounded and natural

## Accessibility

All colour combinations meet WCAG AA contrast requirements when used as specified:
- `#2D3A2D` on `#FDF8F0` — passes AA for body text
- `#E87A4A` on `#FDF8F0` — passes AA for large text and interactive elements
- `#FFFFFF` on `#E87A4A` — passes AA for button labels
