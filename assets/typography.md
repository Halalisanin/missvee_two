# Miss Vee Perfume — Concept Two: Typography

| Usage | Font Family | Style |
|-------|-------------|-------|
| Headings | Playfair Display or Cormorant Garamond | Serif, elegant, botanical feel |
| Body Text | Lato or Nunito | Sans-serif, soft, highly readable |
| Product Names | Playfair Display (italic) | Adds a touch of luxury |
| Buttons/CTAs | Lato (bold) | Clear, actionable |

## Pairing Notes

- **Playfair Display + Lato** — Classic serif/sans-serif pairing. Playfair adds elegance for headings; Lato keeps body text clean and readable.
- **Cormorant Garamond + Nunito** — Softer, more botanical feel. Cormorant Garamond has a delicate, floral quality; Nunito's rounded forms feel friendly and calm.

## Hierarchy

| Element | Font | Size | Weight |
|---------|------|------|--------|
| H1 | Playfair Display | 2.5–3rem | Bold |
| H2 | Playfair Display | 1.75–2.25rem | Semi-bold |
| H3 | Playfair Display | 1.25–1.5rem | Medium |
| Body | Lato / Nunito | 1rem | Regular |
| Product Name | Playfair Display (italic) | 1.125rem | Medium |
| Button | Lato / Nunito | 1rem | Bold |
| Small/Caption | Lato / Nunito | 0.875rem | Regular |
